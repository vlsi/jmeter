# setUp Thread Group

![setUp Thread Group](/img/images/screenshots/setup_thread_group.png)

A special type of ThreadGroup that can be utilized to perform Pre-Test Actions.  The behavior of these threads
    is exactly like a normal [Thread Group](#thread-group) element.  The difference is that these type of threads
    execute before the test proceeds to the executing of regular Thread Groups.
