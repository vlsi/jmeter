# MD5Hex Assertion

![MD5Hex Assertion](/img/images/screenshots/assertion/MD5HexAssertion.png)

The MD5Hex Assertion allows the user to check the MD5 hash of the response data.


### Properties

| Property | Required | Description |
| --- | --- | --- |
| Name |  | Descriptive name for this element that is shown in the tree. |
| MD5 sum | Yes | 32 hex digits representing the MD5 hash (case not significant) |
