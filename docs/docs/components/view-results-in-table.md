# View Results in Table

![View Results in Table](/img/images/screenshots/table_results.png)

[View Results Tree](#view-results-tree), this visualizer uses a lot of memory.By default, it only displays the main (parent) samples; it does not display the sub-samples (child samples).
JMeter has a "`Child Samples?`" check-box.
If this is selected, then the sub-samples are displayed instead of the main samples.
